# SANGBYTE VUE COMPONENTS

This library created using [Day.js](https://day.js.org/), [Maska](https://beholdr.github.io/maska/#/) and [PDF.js](https://mozilla.github.io/pdf.js/).

## included components:
- Accordion
- Alert
- Bar Progress
- Breadcrumb
- Button
- Button Group
- Card
- Carousel
- Circular Progress
- Datatable
- File Preview
- Modal
- Stepper
- Tooltip
- Tree View

## included form components:
- Datepicker
- Dropdown
- Checkkbox
- File Input
- Input
- Radio
